# Github init

## Create repository
Create github repository via web page

## Clone to local machine
You can clone the repository to your local machine (e.g. into your projects folder). It will create a directoy and put the content there
<code>
git clone git@github.com:stefan-11/github-test.git
</code>
This retrieves the repository github-test from github and creates a directory named "github-test" in the folder where you enter the command.

## Commit changes locally
<code>
git add *
git commit -m "enter a commit message here"
</code>

## Push changes to github repository
enter the repository directory. From there enter
<code>
git push
</code>
